# Historical maps of Chicago locations

Collections of historical maps and aerial imagery, each covering a single Chicago location. Every collection folder contains the source files named `{year}_{source}_{id}.{ext}` (so they sort chronologically) plus an `index.md` contact sheet describing each file, the coordinates used, and anything that was unavailable.

## Sources

Each collection draws from the same set of public sources, where available for the location:

- **Sanborn fire insurance maps (1989 set)** - City of Chicago
- **80 Acres maps** - City of Chicago
- **Sidwell tax maps** - City of Chicago
- **ISGS Illinois Historical Aerial Photography (1938)** - Illinois State Geological Survey
- **CMAP/NIPC aerials (1970, 1975, 1980, 1985, 1990, 1995)** - Chicago Metropolitan Agency for Planning; georeferenced via `.jgw` world files (EPSG:3435)
- **Cook County orthos (1998 and 2025)** - Cook County GIS
- **Nearmap imagery screenshots** - Cook County's Nearmap viewer

These files are for research use. Attribution belongs to the City of Chicago, Cook County, CMAP, and the Illinois State Geological Survey.
